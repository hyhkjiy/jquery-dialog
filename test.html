弹窗参数:
width: 弹窗宽度
width: 弹窗高度
title: 弹窗标题
content: 弹窗内容
hideTime: 自动关闭的时间(单位:ms),为零则不自动关闭
open: 打开弹窗后的回调函数,可接受一个dialog对象
close: 关闭弹窗后的回调函数,可接受一个dialog对象
showClose: 是否显示关闭按钮,默认为true
enterBtn/cancelBtn: 确认关闭按钮,接受如下json
{
    text: 按钮名称,
    style: 自定义按钮样式的类名,
    cb: 按钮click事件的回调, 可接受一个dialog对象
}
提示框参数:
icon: 图标,可选值有 success/error/info/help
fontColor: 提示文字的颜色
hideTime: 自动关闭的时间(单位:ms),为零则不自动关闭
